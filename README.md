# 🌐 Kerala Ministry Website

A simple and informative static website showcasing details about the Kerala Ministry, including ministers, profiles, and related information. This project is built using HTML and CSS with a focus on clarity and accessibility.

---

## 🚀 Live Website

🔗 https://treesa-prog.github.io/kerala-ministry-website/

---

## 📌 Project Overview

This website provides structured information about the Kerala government ministry and its members. It is designed as a beginner-friendly web project to demonstrate fundamental front-end development skills.

---

## ✨ Features

* 🏠 Home page with introduction
* 👤 Individual minister profile pages
* 📂 Organized navigation between pages
* 📞 Contact page
* 🎨 Clean and simple UI
* 📱 Basic responsive design

---

## 🛠️ Technologies Used

* HTML5 – Structure of the website
* CSS3 – Styling and layout
